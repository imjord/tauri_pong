# Tauri Pong

Pong in Tauri 

![pong game made in tauri](image.png)

# Usage

You will need node npm installed. Also Cargo/Rust if you want to build it.

-  Clone Repo.
-  Npm I in the root dir. 
-  Npm run tauri dev.
-  Enjoy first to 5 wins.
